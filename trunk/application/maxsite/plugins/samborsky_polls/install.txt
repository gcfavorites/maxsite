/*************
* ГОЛОСОВАНИЯ
**************/
1. Заливаем папку samborsky_polls в /application/maxsite/plugins/

2. В админке MaxSite CMS

		Настройки => Плагины => Находим плагин samborsky_polls => Включить
		
3. В том месте шаблона, в котором должно быть выведено голосование добавляем код:

	<? if(function_exists('samborsky_polls')) echo samborsky_polls() ?>
	
4. Переходим в настройки плагина и создаем голосования

/*************
* АРХИВЫ ГОЛОСОВАНИЙ
**************/

1. Открываем файл index.php вашей темы, вставляем примерно следущие строки:

	# Архивы голосований
	if( 'polls-archive' == mso_segment(1) ){
		
		require($type_dir . 'polls-archive.php');
	}
	
2. Создаем файл polls-archive.php в папке type вашей темы

3. Открываем файл polls-archive.php и вставляем в него содержимое файла page.php либо category.php (как удобно), вырезаем все лишнее и вставляем строку

	<? if(function_exists('samborsky_polls_archive')) echo samborsky_polls_archive() ?>

4. Переходим в админку голосований, в раздел "Настройки", задаем УРЛ на архив голосований, к примеру

	/polls-archive/
	
	или
	
	http://rybalka.tv/polls-archive/
